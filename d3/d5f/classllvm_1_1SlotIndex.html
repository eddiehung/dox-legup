Enumerator
Slot_Block 

Basic block boundary. Used for live ranges entering and leaving a block without being live in the layout neighbor. Also used as the def slot of PHI-defs.

Slot_EarlyClobber 

Early-clobber register use/def slot. A live range defined at Slot_EarlyCLobber interferes with normal live ranges killed at Slot_Register. Also used as the kill slot for live ranges tied to an early-clobber def.

Slot_Register 

Normal register use/def slot. Normal instructions kill and define register live ranges at this slot.

Slot_Dead 

Dead def kill point. Kill slot for a live range that is defined by the same instruction (Slot_Register or Slot_EarlyClobber), but isn't used anywhere.

Slot_Count
