Detailed Description

TypeSet - This is either empty if it's completely unknown, or holds a set of types. It is used during type inference because register classes can have multiple possible types and we don't know which one they get until type inference is complete.

TypeSet can have three states: Vector is empty: The type is completely unknown, it can be any valid target type. Vector has multiple constrained types: (e.g. v4i32 + v4f32) it is one of those types only. Vector has one concrete type: The type is completely known.
