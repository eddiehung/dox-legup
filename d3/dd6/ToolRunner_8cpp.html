Find a named executable.

Prepend the path to the program being executed to ExeName, given the value of argv[0] and the address of main() itself. This allows us to find another LLVM tool if it is built in the same directory. An empty string is returned on error; note that this function just mainpulates the path and doesn't check for executability.
