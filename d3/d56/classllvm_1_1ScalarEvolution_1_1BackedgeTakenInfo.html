const SCEV * ScalarEvolution::BackedgeTakenInfo::getExact ( ScalarEvolution *  SE) const

getExact - Return an expression indicating the exact backedge-taken count of the loop if it is known, or SCEVCouldNotCompute otherwise. This is the number of times the loop header can be guaranteed to execute, minus one.

getExact - Get the exact loop backedge taken count considering all loop exits. A computable result can only be return for loops with a single exit. Returning the minimum taken count among all exits is incorrect because one of the loop's exit limit's may have been skipped. HowFarToZero assumes that the limit of each loop test is never skipped. This is a valid assumption as long as the loop exits via that test. For precise results, it is the caller's responsibility to specify the relevant loop exit using getExact(ExitingBlock, SE).
