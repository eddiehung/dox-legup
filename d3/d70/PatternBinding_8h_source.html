7 // Find patterns of instructions and represent them as Pattern Graphs (Graph.h)
8 // See Paper and slides for more information on the algorithm:
9 // S. Hadjis, A. Canis, J.H. Anderson , J. Choi , K. Nam, S. Brown, T.
10 // Czajkowski, "Impact of FPGA Architecture on Resource Sharing in
11 // High-Level Synthesis," ACM/SIGDA International Symposium on Field
12 // Programmable Gate Arrays (FPGA), Monterey, CA, February 2012.
13 // Steps:
14 // 1. Fill a PatternMap container (see PatternMap.h) with all graphs in
15 // the function,
16 // 2. Determine which can be paired together and bound to the same
17 // functional units,
18 // 3. Select a matching of Graphs (pair them up)
19 //
20 //===------------------------------------------------------------------===//
