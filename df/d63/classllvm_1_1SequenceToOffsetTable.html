Detailed Description

template<typename SeqT, typename Less = std::less<typename SeqT::value_type>>
class llvm::SequenceToOffsetTable< SeqT, Less >

SequenceToOffsetTable - Collect a number of terminated sequences of T. Compute the layout of a table that contains all the sequences, possibly by reusing entries.

Template Parameters
SeqTThe sequence container. (vector or string).
LessA stable comparator for SeqT elements.
