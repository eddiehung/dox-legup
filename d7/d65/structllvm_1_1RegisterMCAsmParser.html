Detailed Description

template<class MCAsmParserImpl>
struct llvm::RegisterMCAsmParser< MCAsmParserImpl >

RegisterMCAsmParser - Helper template for registering a target specific assembly parser, for use in the target machine initialization function. Usage:

extern "C" void LLVMInitializeFooMCAsmParser() { extern Target TheFooTarget; RegisterMCAsmParser<FooAsmParser> X(TheFooTarget); }
