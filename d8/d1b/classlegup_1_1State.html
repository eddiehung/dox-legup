Detailed Description

Legup State Representation.

State - Represents one state of a finite state machine. Composed of a sequential list of instructions. Transitions are represented as follows: Every state should have a default next state transition. The rest of the transitions depend on a single LLVM variable. This variable is checked against the LLVM value associated with each transition to determine the next state
