Detailed Description

A forward iterator which reads non-blank text lines from a buffer.

This class provides a forward iterator interface for reading one line at a time from a buffer. When default constructed the iterator will be the "end" iterator.

The iterator also is aware of what line number it is currently processing and can strip comment lines given the comment-starting character.

Note that this iterator requires the buffer to be nul terminated.
