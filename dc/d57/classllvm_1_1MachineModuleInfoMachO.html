DenseMap<MCSymbol*, StubValueTy> llvm::MachineModuleInfoMachO::HiddenGVStubs
private

HiddenGVStubs - Darwin '$non_lazy_ptr' stubs. The key is something like "Lfoo$non_lazy_ptr", the value is something like "_foo". Unlike GVStubs these are for things with hidden visibility. The extra bit is true if this GV is external.
