29 /// State - Represents one state of a finite state machine. Composed of a
30 /// sequential list of instructions.
31 /// Transitions are represented as follows:
32 /// Every state should have a default next state transition. The rest of the
33 /// transitions depend on a single LLVM variable. This variable is checked
34 /// against the LLVM value associated with each transition to determine the
35 /// next state
36 /// @brief Legup State Representation

44  std::set<State *> predecessors;
45  State *transitionDefault;
46  Transition() : variable(0), signal(0), transitionDefault(0){};
47  };
