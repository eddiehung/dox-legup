llvm::DiagnosticInfoOptimizationRemark::DiagnosticInfoOptimizationRemark ( const char *  PassName,
const Function &  Fn,
const DebugLoc &  DLoc,
const Twine &  Msg 
)
inline

PassName is the name of the pass emitting this diagnostic. If this name matches the regular expression given in -Rpass=, then the diagnostic will be emitted. Fn is the function where the diagnostic is being emitted. DLoc is the location information to use in the diagnostic. If line table information is available, the diagnostic will include the source code location. Msg is the message to show. Note that this class does not copy this message, so this reference must be valid for the whole life time of the diagnostic.
