Detailed Description

Path iterator.

This is a bidirectional iterator that iterates over the individual components in path. The forward traversal order is as follows:

  • The root-name element, if present.
  • The root-directory element, if present.
  • Each successive filename element, if present.
  • Dot, if one or more trailing non-root slash characters are present. The backwards traversal order is the reverse of forward traversal.

Iteration examples. Each component is separated by ',':

/ => /
/foo => /,foo
foo/ => foo,.
/foo/bar => /,foo,bar
../ => ..,.
C:\foo\bar => C:,/,foo,bar
