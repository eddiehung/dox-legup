Translation between effective and objectfile load address.

Compute the effective load address, from an objectfile virtual address. This is implemented in a format-specific way, to take into account things like PIE/ASLR when doing dynamic disassembly. For example, on Mach-O this would be done by adding the VM addr slide, on glibc ELF by keeping a map between segment load addresses, filled using dl_iterate_phdr, etc.. In most static situations and in the default impl., this returns Addr.

Detailed Description

Disassemble an ObjectFile to an MCModule and MCFunctions. This class builds on MCDisassembler to disassemble whole sections, creating MCAtom (MCTextAtom for disassembled sections and MCDataAtom for raw data). It can also be used to create a control flow graph consisting of MCFunctions and MCBasicBlocks.
