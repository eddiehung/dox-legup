Detailed Description

ExitLimit - Information about the number of loop iterations for which a loop exit's branch condition evaluates to the not-taken path. This is a temporary pair of exact and max expressions that are eventually summarized in ExitNotTakenInfo and BackedgeTakenInfo.

If MustExit is true, then the exit must be taken when the BECount reaches Exact (and before surpassing Max). If MustExit is false, then BECount may exceed Exact or Max if the loop exits via another branch. In either case, the loop may exit early via another branch.

MustExit is true for most cases. However, an exit guarded by an (in)equality on a nonunit stride may be skipped.
